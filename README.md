# Tretec Quote System

Ett offertsystem för Tretec Larm AB, byggt med Python och Flask.

## Funktioner
- Produkthantering med sök och filtrering
- PDF-offertgenerering med företagsprofil
- Ellås-produkter från Låsgiganten (automatiskt skrapade)
- Rabattsystem för produkter och kunder
- Tjänster med timbaserade beräkningar
- Serverbaserad PDF-generering

## Installation
1. Klona repositoryt
2. Installera dependencies: `pip install flask flask-cors reportlab requests beautifulsoup4`
3. Kör servern: `python server.py`
4. Öppna http://localhost:5000
